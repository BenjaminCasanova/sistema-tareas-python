
# Sistema de Registro de Tareas Personales

Este proyecto consiste en un programa desarrollado en Python que permite gestionar tareas personales mediante un menú interactivo.

Funcionalidades

 Agregar tareas
 Listar tareas
 Marcar tareas como completadas
 Eliminar tareas

 Tecnologías utilizadas

 Python

 Descripción

El sistema permite al usuario administrar una lista de tareas, donde cada tarea tiene un nombre y un estado (Pendiente o Completada).
