# Investigación Complementaria

## 1. ¿Qué es Git?

Git es una herramienta que sirve para guardar los cambios que hacemos en un proyecto. Gracias a Git podemos volver a una versión anterior si cometemos un error y también llevar un registro de todo lo que hemos ido modificando.

---

## 2. ¿Qué es GitHub?

GitHub es una plataforma donde podemos guardar nuestros proyectos de Git en Internet. Además, permite compartir el código con otras personas y trabajar en equipo de forma más ordenada.

---

## 3. ¿Cuál es la diferencia entre Git y GitHub?

La diferencia es que Git es el programa que usamos para controlar las versiones del proyecto desde nuestro computador, mientras que GitHub es la página donde podemos subir esos proyectos para guardarlos en la nube o compartirlos con otras personas.

---

## 4. ¿Qué es un repositorio?

Un repositorio es la carpeta donde se guarda todo el proyecto junto con su historial de cambios. Ahí quedan registrados los archivos y todas las modificaciones que se han realizado.

---

## 5. ¿Qué función cumple git add?

El comando **git add** sirve para seleccionar los archivos que queremos guardar en el siguiente commit. Es como preparar los cambios antes de registrarlos.

---

## 6. ¿Qué función cumple git commit?

El comando **git commit** guarda los cambios realizados en el proyecto y les agrega un mensaje para indicar qué fue lo que se modificó.

---

## 7. ¿Qué función cumple git push?

El comando **git push** envía los cambios que hicimos en nuestro computador al repositorio que está en GitHub, para que queden respaldados en línea.

---

## 8. ¿Qué función cumple git pull?

El comando **git pull** sirve para descargar los cambios que existen en GitHub y actualizarlos en nuestro computador, manteniendo el proyecto al día.

---

## 9. ¿Cuáles son las ventajas del control de versiones?

Algunas ventajas son:

- Permite recuperar versiones anteriores del proyecto.
- Guarda un historial de todos los cambios.
- Facilita el trabajo en equipo.
- Ayuda a evitar perder información importante.
- Hace más fácil organizar el desarrollo de un proyecto.

---
